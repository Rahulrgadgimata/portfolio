#Portfolio 🌌

A modern, high-impact personal portfolio website built with an **Antigravity** aesthetic. Designed for AI Engineers and Technical Leaders.

## 🚀 Features
- **🌌 Google Confetti Engine:** A custom JavaScript particle system featuring Google's signature color palette.
- **🌊 Wave-Animated Icon Strip:** Floating circular tech icons with staggered wave motion.
- **💎 Glassmorphic UI:** Premium frosted glass card components with backdrop-blur effects.
- **⌨️ Dynamic Typing Effect:** Rotating professional headlines in the hero section.
- **📱 Fully Responsive:** Optimized for all screen sizes from mobile to ultra-wide displays.
- **🛡️ SVG-Powered Icons:** Professional, emoji-free iconography for a clean engineering look.

## 🛠️ Tech Stack
- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Vanilla, Variables, Keyframe Animations)
- **Logic:** Vanilla JavaScript (ES6+)
- **Theme:** Google Antigravity Inspired

## 📂 Project Structure
- `index.html`: The core structure and content.
- `style.css`: The design system and animations.
- `script.js`: Interactive elements and particle engine.
